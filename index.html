Privacy Policy
---------------
Last Update      : January 12,2023
Application Name : SoundTester
Publisher        : Erol Kucuk
Contact          : erolkucuk@gmail.com

The application does not collect,store or share any personal data from its users.
The application only stores application specific settings preferences. 
These preferences can be seen under settings menu option and are stored locally on your device.

This application gets audio data from the device microphone. 
Because of this it needs the android.permission.RECORD_AUDIO permission. 
This permission allows application record audio data.
Audio data got from the microphone is temporarily kept in the device memory and cleared after application closed and is never stored or shared.
