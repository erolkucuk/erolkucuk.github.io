SOUNDTESTER

Settings

Microphone: Microphone to be used to record audio. It can be built-in microphone,wired or bluetooth headset microphone.

Duration: Audio record buffer size in seconds.Default is 1s.Maximum duration size is limited to 10s.

Sample rate: Input audio data's sampling rate. Default is 44100. Maximum rate is limited to 192000.

Fft number: Fft algorithm uses 2^(fft number) number samples to calculate fourier transform.Default is 13 (2^13=8192).Maximum fft number is limited to 15.


Toolbar/Menu Commands

Start/Stop: Starts and stops recording.

Info: Opens and closes info panel.

Settings: Opens settings page.

Normalize: Normalizes audio waveform.It works if recording is already stopped.

Reset Zoom: Resets zoom ratios to 1.0.

Privacy Policy: Shows application privacy policy document.

Help: Shows offline application manual.

Descriptions

-This application gets audio data from microphone and calculates its fourier transform as a frequency spectrum.
Fourier transform converts a time domain signal to frequency-domain signal. In other words fourier transform finds which frequency components the input signal consists of. Fft(Fast fourier transform) is an algorithm to calculate a time domain signal's fourier transform with computational efficiency.
Sampling rate of input data determines resulting frequency spectrum range so
if sample rate = 44100 then frequency spectrum is between 0 and 22050Hz(sample rate/2).

-To start/stop recording audio press "MIC" button on the toolbar. Application will record audio only last as many seconds as "Duration".

-There is two graphs in the app, first(audio graph) shows input waveform and second(fft graph) shows fourier transform result as frequency spectrum.

-Yellow bar on the audio graph shows which data section to be used as input for fft analysis. This bar shows up only at zoom level 1. User can slide this bar left and right to choose data section.

-Graphics can be dragged and zoomed in/out by pinching on data or sliding fingers on axes. When zooming ,involved axis labels colors will change to yellow and +/- signs will appear. Zoom level ratio also will be printed on the graph while zooming.

-Fft frequency spectrum resolution is (sample rate)/2^(fft number). Increasing fft number will increase accuracy of spectrum but will cause more cpu usage.

-Info panel shows actual applied settings for audio sampling and fft analysis. Also lists frequency spectrum's most dominant lines. This list is limited to 16 items. To open/close info panel click (i) button on the toolbar. Alternatively user can close info panel by swiping left on it or tapping info panel title. Fft data graphic also prints most dominant frequency in the audio signal.
